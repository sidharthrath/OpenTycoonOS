# Changelog

## 0.1.0-alpha.0

Initial public alpha of OpenTycoonOS.

- Published the core TypeScript simulation package as `opentycoonos`.
- Included ESM, CommonJS, and TypeScript declaration builds through `tsup`.
- Added public docs for architecture, roadmap, game compositions, contribution, security, and agent-driven game building.
- Added `AGENTS.md`, `llms.txt`, and `docs/module-catalog.json` for coding-agent discoverability.
- Added a minimal service-game example.

OpenTycoonOS was extracted from private prototype work into a clean public repository. The public repo intentionally excludes private game prototypes and studio workflow files.
